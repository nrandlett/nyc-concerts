# lib/ticketmaster

Client code for the [Ticketmaster Discovery API](https://developer.ticketmaster.com/products-and-docs/apis/discovery-api/v2/).
This will fetch upcoming events for NYC and filter them down to small venues
(capacity under 2,500), then normalize them into our own event shape before
they get cached in the database.
