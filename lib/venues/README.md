# lib/venues

Custom scrapers for our curated list of ~40 small NYC venues that don't appear
on Ticketmaster. Each scraper fetches a venue's upcoming shows and normalizes
them into the same event shape used everywhere else in the app.

The list of venues to scrape is seeded from [`data/venues.json`](../../data/venues.json).
